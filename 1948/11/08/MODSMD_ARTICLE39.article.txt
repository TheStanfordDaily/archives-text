# Exams Now Ready In Reserve Room
## 
### 
Examination files will be avail
able for reference in the Reserve
Book Room of the main library be
ginning today. Exams will be on
two-hour reserve.

Honor Code subcommittee for exam
files, said that the files are not yet
complete, but more exams will be
added as soon as possible.
Purpose of the exam files is to
make the exams available to all
students. In ihc past, only a small
part of the student body has had
access to such files.
All examinations in the library
will be listed in the card file in the
